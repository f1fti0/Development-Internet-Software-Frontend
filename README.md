# Разработка интернет-приложений (РИП) 2025

## Расчет времени миграции данных

Разработанная система позволит инженерам данных создавать и формировать заявки на расчёт времени миграции данных для выбранных ими методов, а диспетчерам миграций — завершать или отклонять данные заявки.
